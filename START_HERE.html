Portflow Exporter

Welkom! Dit is de interactieve gids om het visuele Portflow dashboard op jouw eigen computer te starten. Kies hieronder de route die het beste bij jouw werkwijze past.

1 Het Dashboard Starten

Kies één van de onderstaande routes om de lokale server op jouw computer te lanceren. Het inloggen doe je daarna gewoon veilig via de browser-interface!

Route A: De Designer (VS Code / IDE)

Dit is de makkelijkste route als je Visual Studio Code (VS Code) al gebruikt of gewoon simpel de map open hebt.

  1. Sleep je volledige portflow map in VS Code om het project te openen.
  2. Ga in het bovenste menu van VS Code naar Terminal → New Terminal.
  3. Typ het volgende commando onderin je scherm en druk op Enter:
python3 portflow_export.py

Het dashboard opent nu automatisch in je internetbrowser!

Route B: De Developer (GitHub User / Mac Terminal)

Voor gebruikers die comfortabel zijn met de terminal of command line.

  1. Open de Terminal (of iTerm2).
  2. Navigeer naar de map waar je het project hebt geplaatst:
cd ~/Downloads/portflow

3. Installeer indien nodig de vereisten, en start de lokale server:

pip install -r requirements.txt
python3 portflow_export.py

Route C: De Standaard Windows Gebruiker

Als je op een Windows laptop werkt zonder speciale code-editors of terminal ervaring.

  1. Druk op de Windows-toets, typ cmd en druk op Enter om de Command Prompt te openen.
  2. Navigeer naar de map waar je portflow hebt uitgepakt (bijv. in je Downloads):
cd Downloads\portflow

3. Start het script met Python (zorg dat Python 3 is geïnstalleerd vanuit de Microsoft Store!):

python portflow_export.py

Klaar! 🎉

Zodra het script draait, springt je browser vanzelf open naar http://127.0.0.1:8080.

Om de applicatie later weer veilig af te sluiten, ga je terug naar je terminal of Command Prompt en druk je op Ctrl + C.
